# Originality Guardrails

## Purpose
Prevent copyright and fidelity issues.

## Prompt Block
Create structurally faithful but fully original training material.

Do not:
- copy official TestDaF texts, graphics, audio scripts, or solutions
- paraphrase official demo tasks too closely
- claim that generated content is an official exam item

Do:
- preserve public task structure and difficulty intent
- use fresh topics and source material
- signal uncertainty when acoustic realism is limited in text-first mode
